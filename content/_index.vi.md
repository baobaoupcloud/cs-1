# Giới Thiệu về Lập Trình C

Chào mừng bạn đến với Tuần 1 của khóa học CS50! Module này sẽ giới thiệu các kiến thức nền tảng về lập trình sử dụng ngôn ngữ C.

#### Mục Tiêu Học Tập

Kết thúc module này, bạn sẽ có khả năng:

* Viết được các chương trình cơ bản bằng ngôn ngữ C
* Hiểu được các khái niệm cốt lõi trong lập trình
* Làm việc với biến và kiểu dữ liệu
* Triển khai logic điều kiện
* Tạo vòng lặp cho các tác vụ lặp đi lặp lại
* Sử dụng các toán tử và hàm

#### Nội Dung Workshop


#### 1. Bắt Đầu với C

* Tổng quan khóa học
* Cài đặt môi trường development
* Chương trình C đầu tiên của bạn

#### 2. Nền Tảng Lập Trình

* Cú pháp cơ bản
* Cấu trúc chương trình
* Quy trình compilation

#### 3. Làm Việc với Biến

* Data types (kiểu dữ liệu)
* Variable declaration (khai báo biến)
* Khái niệm về memory (bộ nhớ)

#### 4. Control Flow với Câu Điều Kiện

* If statements (câu lệnh if)
* Switch cases
* Boolean logic (logic Boolean)

#### 5. Triển Khai Vòng Lặp

* For loops
* While loops
* Do-while loops

#### 6. Sử Dụng Toán Tử và Hàm

* Arithmetic operators (toán tử số học)
* Logical operators (toán tử logic)
* Function declaration và usage (khai báo và sử dụng hàm)

#### 7. Thực Hành

* Coding exercises (bài tập lập trình)
* Problem sets (bộ bài tập)
* Debug challenges (thử thách gỡ lỗi)

#### Phân Bổ Thời Gian

* Tổng thời gian Workshop: ~2 giờ
* Lý thuyết: 45 phút
* Bài tập thực hành: 75 phút

#### Yêu Cầu Tiên Quyết

Trước khi bắt đầu workshop này, bạn cần có:

* Máy tính đã cài đặt C compiler
* Hiểu biết cơ bản về thao tác máy tính
* Text editor hoặc IDE tùy chọn (tham khảo [Visual Studio Code](https://code.visualstudio.com/))
