# Introduction to C Programming

Welcome to Week 1 of CS50! This module introduces you to the fundamentals of programming using the C language.

#### Learning Objectives

By the end of this module, you will be able to:

* Write basic programs in C
* Understand core programming concepts
* Work with variables and data types
* Implement conditional logic
* Create loops for repetitive tasks
* Use operators and functions

#### Workshop Contents


#### 1. Getting Started with C

* Course overview
* Development environment setup
* Your first C program

#### 2. Programming Fundamentals

* Basic syntax
* Program structure
* Compilation process

#### 3. Working with Variables

* Data types
* Variable declaration
* Memory concepts

#### 4. Control Flow with Conditionals

* If statements
* Switch cases
* Boolean logic

#### 5. Implementing Loops

* For loops
* While loops
* Do-while loops

#### 6. Using Operators and Functions

* Arithmetic operators
* Logical operators
* Function declaration and usage

#### 7. Hands-on Practice

* Coding exercises
* Problem sets
* Debug challenges

#### Time Allocation

* Total Workshop Time: ~2 hours
* Theory: 45 minutes
* Hands-on Exercises: 75 minutes

#### Prerequisites

Before starting this workshop, you should have:

* A computer with a C compiler installed
* Basic understanding of computer operations
* Text editor or IDE of your choice (consider [Visual Studio Code](https://code.visualstudio.com/))

{{% notice note %}}
This workshop is part of the CS50 course series. While no prior programming experience is required, familiarity with basic computer operations is helpful.
{{% /notice %}}
